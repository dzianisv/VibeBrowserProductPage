## One-Sentence Description

The AI browser that acts, not just answers - autonomous browser automation with any AI model, running fully private on your device.

## Short Description (75 chars max)

AI browser agent that automates web tasks using natural language commands

## Value Proposition

Vibe Browser is an AI-powered browser extension that acts as an autonomous agent, operating the web on behalf of users. Complete autopilot for the web.

## Three Core Use Cases

1. **Outreach & Lead Gen**: LinkedIn, Twitter, Reddit outreach with profile analysis, personalization, and draft messages you approve before sending.

2. **Deep Research**: Reads across dozens of sources, synthesizes into recommendations, outputs tables and next steps.

3. **Web Automation**: Navigate legacy sites, handle forms, filters, carts. Stops before payment - you decide.

## Key Capabilities

- Natural language task definition: Users describe what they need in plain English
- Autonomous web navigation: Vibe searches, browses, and interacts with websites
- Multi-site workflows: Handles complex tasks across LinkedIn, Twitter, Gmail, Reddit, and any website
- Privacy-first: Supports local AI (Gemini Nano) so data never leaves the device
- Model agnostic: Works with GPT, Claude, Gemini, Grok, DeepSeek - switch anytime
- User stays in control: Review and approve every action, nothing sent without confirmation

## Unique Differentiators

**The only AI browser with:**

1. **True local AI** - Gemini Nano runs 100% on-device, data never leaves your machine
2. **Any LLM support** - No vendor lock-in, use GPT, Claude, Gemini, Grok, DeepSeek
3. **Universal website compatibility** - Works on LinkedIn, Twitter, Reddit, Gmail, any site. No APIs needed
4. **Privacy-first architecture** - TEE (Trusted Execution Environment) coming for enterprise security
5. **Works as extension** - Install on any Chrome browser, no special browser fork required
6. **Google Workspace Integration** - Native OAuth 2.0 API access to Gmail and Calendar
7. **Skills Standard** - Supports agentskills.io portable instruction format
8. **MCP Server & Client** - Can be controlled by external agents AND connect to external MCP servers
