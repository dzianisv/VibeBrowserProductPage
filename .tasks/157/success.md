# Success Metric

Task is done when **opening `https://www.vibebrowser.app/cli` and `https://www.vibebrowser.app/mcp` in a fresh browser** produces **clear, copyable setup flows that explain the remote relay and direct MCP endpoint without Studio or a `vibebrowser-mcp` install** in **the live Vibe Browser product site**.

Not done at: CI green, unit tests passing, a preview deployment, or an implementation report.
