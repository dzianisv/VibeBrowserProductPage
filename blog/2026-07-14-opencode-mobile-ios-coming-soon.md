We're building an iOS version of OpenCode Mobile. It's not released yet — we're actively working on it and it's coming soon.

OpenCode Mobile is a phone app that connects to your own [opencode server](https://opencode.ai) — the same one you run on your laptop or a home machine. From the app you can see your coding sessions, send it new tasks, watch replies stream in as the agent writes them, review file diffs, and approve or reject tool calls. It's a remote control for an agent that's already doing the work, not a new agent on its own.

The Android version is already available to testers and works the same way — connect to your server, manage sessions, review diffs, approve tools, all from your phone. If you're on Android today, you can get it on [Google Play](https://play.google.com/store/apps/details?id=cc.agentlabs.opencode).

## What to expect on iOS

Same idea, same server, same workflow — built for iPhone and iPad. No new backend, no new account system. You point it at your existing opencode server and it does what the Android app does.

## Staying open source

OpenCode Mobile is open source today and will stay open source on iOS. The code lives at [github.com/dzianisv/opencode-mobile](https://github.com/dzianisv/opencode-mobile) — you can read it, build it, or file an issue.
