- Send Tues-Thurs, 10am-2pm PT

- Personalize each email
- Follow up 2-3 times
- Keep under 200 words
- Use plain text, no attachments

**DON'T:**
- Send from @gmail.com if you have @vibebrowser.app
- Give up after one email
- Attach files (use links)
- Write long essays
- Send outside optimal hours
